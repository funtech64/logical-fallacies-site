False Consensus Effect

A logical fallacy

The false consensus effect occurs when someone overestimates the extent to which their beliefs or opinions are typical of those of others. This fallacy shifts the focus from sound reasoning to unfounded assumptions about consensus.

For example, "Everyone I know thinks this way, so it must be true" is an example of the false consensus effect because it incorrectly assumes that one's personal beliefs are universally held.

Critical thinking involves evaluating arguments based on evidence and logical reasoning, rather than relying on assumptions about consensus. By doing so, we can avoid the false consensus effect and maintain a focus on the issues at hand.

See also: Subjectivist Fallacy

Examples

Business Example: "Everyone in our team agrees with this decision, so it must be the right one."

Academic Example: "All my colleagues think this theory is correct, so it must be valid."

Pop-Culture Example: "In TV shows, characters often use the false consensus effect to justify their beliefs."

Intimate Partner Example: "All my friends think you're wrong, so you must be."
